# [ USRT ]
#### Create Google like short url
###### Installation
  - Download or clone the repository
  - Put inside web accessible directory ( ex. /var/www/html )
  - Import usrt.sql file
  - Edit /src/config/config.php and set the database information

###### Technology Used

  - PHP7
  - Mysql (PDO)
  - Jquery
  - Vue.js
  - Bootstrap
